# Bookmark Manager
This project is a simple and responsive Bookmark Manager that allows users to save their favorite websites, visit them directly, and delete them when needed. It also includes input validation to ensure correct website URLs and a clean user interface built using Bootstrap and FontAwesome.

# Technologies Used
HTML
CSS
Bootstrap 5
JavaScript
FontAwesome

# Features
Add new bookmarks
Delete bookmarks
Visit website using a "Visit" button
Validation for empty fields and invalid URLs
Responsive design using Bootstrap
Clean UI with FontAwesome icons
